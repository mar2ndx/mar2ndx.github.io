First thing after getting a new Mac:

```bash
npm install -g pnpm
```

## pnpm global

```bash
pnpm install -g @tarojs/cli
or
pnpm add -g @tarojs/cli
```

for npm:

```bash
npm install -g @tarojs/cli
```

# Installed packages

```bash
pnpm list -g --depth=0
```

For npm:

```bash
npm list -g --depth=0
```
