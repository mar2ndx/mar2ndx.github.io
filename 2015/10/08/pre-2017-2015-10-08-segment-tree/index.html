[Fundamental] Segment Tree
2015 October 8th

Overview

Segment tree is a tree data structure for storing intervals, or segments.

Can be used to search the max/min or sum values in a range.

  1. modify = O(log n)

  2. query = O(log n)

  3. build = O(n)
